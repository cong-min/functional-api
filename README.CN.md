<h1 align="center">Functional API</h1>

<p align="center">
  基于 Koa 和 TypeScript 的<strong>函数式 API</strong> node.js 服务</a>
</p>

## 特性

- API 路由规则指向 function 函数，无需配置路由，函数即服务
- 支持 TypeScript runtime，无需手动编译，启动脚本即直接运行应用
- 封装启动应用服务的入口文件，聚焦至业务代码
